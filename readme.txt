== Description ==

TwitterSifu support and tested on both latest standard Wordpress and Wordpress MU. 

TwitterSifu is a plugin designed to tweet whenever you publish a new post and also your older posts to get more traffic. As for the older posts the plugin will randomly picks your older post based on the interval specified by you. The primary function of this plugin is to promote older blog posts by tweeting about them and getting more traffic.

== Installation ==

1. Download the latest version of the TwitterSifu Plugin to your computer.
2. With an FTP program, access your site’s server.
3. Upload the Plugin folder to the /wp-content/plugins/twittersifu.
4. In your WordPress Administration Panels, click on Plugins from the menu.
5. You should see "TwitterSifu" Plugin listed. If not, with your FTP program, check the folder to see if it is installed. If it isn’t, upload the file(s) again. If it is, delete the files and upload them again.
6. To turn the TwitterSifu Plugin on, click Activate.
7. Check your Administration Panels or WordPress blog to see if the Plugin is working. Once activated you shall see the TwitterSifu appear on your left hand sidebar.
8. You can change the plugin options from TwitterSifu under settings menu.

Alternatively you can also follow the following steps to install the TwitterSifu plugin:

1. In your WordPress Administration Panels, click on Add New option under Plugins from the menu.
2. Click on upload at the top.
3. Browse the location and select the TwitterSifu Plugin and click install now.
4. To turn the TwitterSifu Plugin on, click Activate.
5. Check your Administration Panels or WordPress blog to see if the Plugin is working.
6. You can change the plugin options from TwitterSifu under settings menu.
